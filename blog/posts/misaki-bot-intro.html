Memperkenalkan Misaki Bot - WhatsApp Bot Multifungsi

18 Februari 2025 | oleh EmuVlucht

Hai semuanya! Hari ini aku mau memperkenalkan project yang sudah lama aku kerjakan - Misaki Bot.

Misaki Bot adalah WhatsApp bot yang didesain untuk membantu komunitas dengan berbagai fitur menarik. Bot ini dibangun dengan JavaScript dan menggunakan teknologi modern seperti Prisma ORM untuk database management.

Fitur Utama

  • Moderation Tools - Ban, kick, mute, dan warning system untuk menjaga ketertiban grup
  • Fun Commands - Games, anime reactions, dan trivia untuk menghibur member
  • Utility - Info grup, user lookup, reminder, dan berbagai tools berguna lainnya
  • Customizable - Prefix dan settings bisa dikustomisasi sesuai kebutuhan grup

Tech Stack

  • JavaScript - Bahasa pemrograman utama (89.4% of codebase)
  • Prisma ORM - Database management yang type-safe
  • whatsapp-web.js - Library untuk berinteraksi dengan WhatsApp API
  • Node.js - Runtime environment

Open Source

Project ini sepenuhnya open source dan dilisensikan dengan dual license (MIT + BudzVerse Labs Custom License). Kamu bebas menggunakan, memodifikasi, dan mendistribusikan kode ini selama mengikuti ketentuan lisensi.
